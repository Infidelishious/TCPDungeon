# TCP Dungeon

Download the zip and explore the dungeon below TCP!

All .class files that are classes that extend Room 
and are in the same folder will be added to the game!

```
Included example rooms:
  SadRobot.java
  GuessingGame.java
  FastMath.java
```

### To compile: "javac *.java"
### To run: "java Dungeon"
